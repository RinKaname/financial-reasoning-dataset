# Dataset Description

## Overview
This dataset is designed for training models on advanced financial reasoning. It consists of synthetic data collaboratively generated by a human expert and an AI assistant. The dataset covers a wide array of specialized financial domains, providing complex questions along with step-by-step reasoning (Chain-of-Thought) and final expert answers.

## File Structure
- `train/train.jsonl` — Labeled training data containing questions, reasoning, and answers.

## Features
| Column   | Type   | Description |
|----------|--------|-------------|
| question | string | The financial problem or question presented to the model. |
| cot      | string | The Chain-of-Thought reasoning enclosed in `<think>...</think>` tags to solve the problem step-by-step. |
| answer   | string | The final, professional answer or explanation derived from the reasoning process. |

## Notes
**Domain-Specific Context:**
This highly specialized dataset covers advanced topics such as stochastic calculus, machine learning in finance, options pricing, and risk management. It also encompasses professional domains including Venture Capital, Private Equity, Investment Banking (M&A), Restructuring, and Hedge Fund operations. Authentic industry lexicon (e.g., 'Pod Shop', 'GMV', 'Dollar Neutral') is used throughout to ensure high-quality, realistic case studies and expert dialogues.

**Synthetic Data Generation Process:**
The data was synthetically generated through iterative human-AI collaboration to simulate expert-level, deep-reasoning scenarios (e.g., Hedge Fund Portfolio Manager case studies). The generation process prioritized deep, accurate, and educational content ('teaching mode'), explicitly avoiding generic outputs. The dataset was progressively built using Python scripts that generated, formatted, and validated the data to prevent duplication, maintain realistic topic distributions, and ensure appropriate string lengths for both the reasoning (`cot`) and the final `answer`.
